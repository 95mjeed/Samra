# Samra · سَمْره

Audiobook & e-book downloader and player for content you own — Storytel.

A multi-platform project:

- **`android/`** — native Android app (Kotlin + Jetpack Compose). In-app audiobook
  player (chapters, speed, sleep timer, resume + bookmarks) and an EPUB/PDF reader
  (themes, fonts, highlights/notes, resume + bookmarks). Bilingual (العربية / English).
- **`pc/`** — Windows desktop app (Python + Flet). Library, downloads, in-app audio
  player and EPUB reader with resume + bookmarks, packaged with an installer.
- **`design/`** — brand + design-system assets (app icon, tokens).
- **`ios/`** — placeholder for a future iOS app.

Private project — for personal use with content you have the right to download.
